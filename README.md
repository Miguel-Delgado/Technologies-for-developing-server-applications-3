Это приложение демонстрирует различные механизмы аутентификации, авторизации и работы с базой данных, в контрольной работе №3.  
Включает:
- Базовую HTTP-аутентификацию и хеширование паролей (passlib + bcrypt)
- JWT-аутентификацию и регистрацию пользователей
- Ограничение частоты запросов (rate limiting)
- Ролевую модель доступа (RBAC)
- Защиту документации в зависимости от режима окружения (DEV/PROD)
- CRUD-операции для ресурса «Todo» с SQLite
